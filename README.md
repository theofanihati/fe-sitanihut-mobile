# 🌱 SITANIHUT – Sistem Informasi Petani Hutan - Mobile

Versi mobile (Android) dari platform **Sitanihut**, dibangun dengan **Kotlin** dan **Jetpack Compose**.  Menggunakan **Clean Architecture**, **Material Design 3**, serta **Dagger Hilt** untuk dependency injection.

---

## 🚀 Teknologi yang Digunakan

- [Kotlin: Bahasa pemrograman utama](https://kotlinlang.org/)
- [Jetpack Compose: Framework UI deklaratif](https://www.google.com/search?q=jetpack+compose&rlz=1C1ONGR_enID1032ID1032&oq=jetpack+compose&gs_lcrp=EgZjaHJvbWUqBggAEEUYOzIGCAAQRRg7MgYIARBFGDvSAQgyNTg4ajBqNKgCALACAA&sourceid=chrome&ie=UTF-8)
- [Material Design 3](https://m3.material.io/)
- [Hilt: Dependency Injection](https://dagger.dev/hilt/)
- Clean Architecture (Presentation, Domain, Data)
- Room: Database lokal
- DataStore: Penyimpanan preferensi
- Retrofit: Library HTTP
- [Lottie: Animasi berbasis JSON](https://lottiefiles.com/)

---

## 📦 Instalasi

Pastikan Anda sudah menginstal **Node.js** (disarankan versi 18 ke atas).

### Langkah-Langkah Instalasi

1. Clone repo:
```bash
   git clone <git-url-repo>
```
2. Buka project di Android Studio (Giraffe/lebih baru).
3. Sync Gradle dependencies.
4. Jalankan di emulator atau perangkat Android.
